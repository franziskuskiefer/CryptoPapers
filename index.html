Passwords

Password based cryptography is concerned with cryptography from low-entropy secrets. In contrast to conventional cryptographic algorithms and protocols, secrets used here have low-entropy, such that an adversary is able to iterate through all possible secrets in reasonable time. This leads directly to the inherent threat to everything password related: so-called dictionary attacks.
